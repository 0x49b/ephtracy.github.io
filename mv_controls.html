Shortcut

  • MacOS : Ctrl -> Command
    • Key Command Modifier
    Console
    TAB activate console
    F6 / 6 screenshot +CTRL : for whole window
    Camera
    RButton rotate camera =N+LButton
    RButton + SPACE move camera focus
    RButton + X set camera focus
    WASDQE move/rotate camera
    WHEEL zoom in/out
    F5 / 5 save current camera pose
    F7 / 7 load current camera pose
    Edit
    CTRL+Z undo
    CTRL+Y redo =CTRL+SHIFT+Z
    Model File
    CTRL+S Save model
    CTRL+SHIFT+S Save As
    CTRL+O Open model
    CTRL+SHIFT+O Import Model
    CTRL+N New model
    CTRL+SHIFT+N Duplicate model
    Model Edit
    U fUll model
    I fIll model
    Backspace empty model
    Brush Shape
    V Voxel shape brush +/- : resize brush
    F Face shape brush
    B Box shape brush
    L Line shape brush
    P Pattern shape brush
    Brush Type
    ALT+LButton pick voxel color
    CTRL+LButton move model +SHIFT : 2d version
    LButton [Renderer Mode] pick focus of Depth of Field
    ALT+LButton [Renderer Mode] pick voxel material
    T aTtach brush +SHIFT : temporally switch to Erase Brush
    R eRase brush +SHIFT : temporally switch to Attach Brush
    G paint brush
    Pattern Brush
    +/- rotate around z axis
    DIR/PAGE local offset
    0 reset local offset and orientation
    Mirror
    1 mirror x
    2 mirror y
    3 mirror z
    Loft
    CTRL+1 loft x
    CTRL+2 loft y
    CTRL+3 loft z
    Render
    CTRL+E display Edge
    CTRL+G display Grid
    CTRL+F display Frame
    CTRL+R display gRound
    CTRL+B display Background
    CTRL+W display shadoW
    CTRL+SHIFT+V display Voxel
    CTRL+SHIFT+M advance render Mode
    CTRL+SHIFT+Y display ambient light
    Palette
    CTRL+DRAG duplicate color
    ALT+DRAG swap color
    CTRL+SHIFT+DRAG fill block
    ALT+SHIFT+DRAG fill gradient block
